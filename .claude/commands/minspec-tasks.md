# /minspec-tasks — MinSpec Tasks Phase

Run the **Tasks** phase. Required for T3+.

Break the plan into small, dependency-ordered checkboxes. Each task must be completable in one session and verifiable from outside.

MinSpec tracks progress from these checkboxes — they must remain in standard markdown checkbox format (`- [ ]` / `- [x]`). Arguments: $ARGUMENTS
